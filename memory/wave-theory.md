# 波浪理论 (Elliott Wave Theory)

## 核心概念

### 基本结构：5浪上涨 + 3浪下跌 = 8浪循环

**上涨5浪：**
- 浪1：初步上涨，通常是熊市后的反弹
- 浪2：回调，但不低于浪1起点
- 浪3：通常是最强的一浪，突破浪1高点
- 浪4：回调，形态复杂
- 浪5：最后一涨，常伴随狂热情绪

**下跌3浪（A-B-C）：**
- A浪：下跌开始
- B浪：反弹，但无法突破前高
- C浪：主跌浪

## 关键原则
1. 浪2永远不跌破浪1起点
2. 浪3永远不是最短的一浪
3. 浪4永远不与浪1重叠

## 斐波那契回撤
- 38.2%, 50%, 61.8% 是常见回撤位
- 预测支撑/阻力位

## 应用于BTC
结合BTC减半周期（4年）和波浪理论分析走势
