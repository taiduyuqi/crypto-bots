# 波浪理论完整笔记

## 艾略特波浪理论 (Elliott Wave Theory)

### 核心理念
市场价格走势呈现特定的波浪模式，由5波推动和3波调整组成一个完整的循环。

### 基本结构（8浪循环）

```
上涨趋势（推动浪）：
┌─ 1浪 ─┐┌─ 2浪 ─┐┌─ 3浪 ──┐┌─ 4浪 ─┐┌─ 5浪 ──┐
│   ↑    ││   ↓   ││    ↑    ││   ↓   ││    ↑   │
└───────┘└───────┘└─────────┘└───────┘└─────────┘

下跌趋势（调整浪）：
┌─ A浪 ─┐┌─ B浪 ─┐┌─ C浪 ─┐
│   ↓   ││   ↑   ││   ↓   │
└───────┘└───────┘└───────┘
```

### 波浪特征

**推动浪 (1-5)**
- 1浪：初步上涨，成交量增加
- 2浪：回撤，但不低于1浪起点（常见回撤50%-61.8%）
- 3浪：最强劲，通常最长（1.618倍1浪）
- 4浪：整理，回撤幅度小（通常38.2%）
- 5浪：最后一波，可能出现背离

**调整浪 (A-C)**
- A浪：下跌开始
- B浪：反弹（不超过5浪高点）
- C浪：最后一跌，通常与A浪等长

### 黄金分割比例
- 0.382 (38.2%)
- 0.5 (50%)
- 0.618 (61.8%)
- 1.618 (黄金比例)

### 波浪等级
- 超大循环：数十年
- 大循环：数年
- 中循环：数月
- 小循环：数周
- 微循环：数天
- 次微循环：数小时

### 实用规则
1. 3浪永远不会是最短的推动浪
2. 2浪永远不会超过1浪起点
3. 4浪永远不会与1浪重叠
4. 交替原则：2浪和4浪形态交替

### 识别技巧
- 成交量配合：推动浪成交量增加
- 均线支撑：价格回调不破重要均线
- 形态配合：三角形、楔形等中继形态
- 背离：RSI背离预示反转

### 在加密货币中的应用
- 适合4小时及以上周期
- 结合斐波那契回撤
- 关注整数关口和历史支撑阻力
- 注意USDT mint/burn 资金流向
